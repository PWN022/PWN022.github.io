# 河北工程大学教务 SOAP 服务列表暴露报告

## 漏洞概况

| 项目     | 内容                             |
| -------- | -------------------------------- |
| 漏洞标题 | 教务系统 SOAP 服务列表未授权暴露 |
| 所属单位 | 河北工程大学                     |
| 漏洞类型 | 信息泄露                         |
| 涉及域名 | jwglxxfwpt.hebeu.edu.cn          |
| 发现时间 | 2025-11-03                       |
| 漏洞等级 | 低危                             |

## 漏洞描述

1. 服务接口列表暴露
   - 访问 `https://jwglxxfwpt.hebeu.edu.cn/ws/` 可查看所有SOAP服务接口。
   - 暴露4个核心业务服务及28个内部方法。
   - 包含学生成绩查询(`getXscjList`)、用户登录(`userLogin`)、成绩修改(`updateXscj`)等敏感业务接口。
2. WSDL接口文档泄露
   - 所有服务端点均公开WSDL文档（如`/ws/commonWebs?wsdl`）。
   - 暴露完整的接口数据结构和参数格式。
   - 攻击者可据此构造精确的SOAP请求。

3. 潜在未授权访问风险

   - 部分接口返回认证错误而非404，表明接口存在但需要认证。

   - 攻击者可基于暴露的接口信息进行进一步渗透测试。
