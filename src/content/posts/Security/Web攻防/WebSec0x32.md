# 隐私合规-判断规则&检测项目

对象：APP 小程序等

当APP、小程序在运行的时候，如果需要获取相关权限(例如位置、相册、通讯录 等等)时，需要以弹窗的形式询问用户的意见，用户有权同意或者拒绝。如果不通知用户就非法获取这些权限，就属于违规。

具体：后续APP安全课程

# 资源拒绝服务-加载受控&处理受控

## 功能1：验证码或图片显示自定义大小

存在可控验证码或者图片大小的参数，比如：

```
ip:port/xxx.php?w=100000&h=100000
```

这样就会给目标站点的中间件以及服务器资源造成影响，也就是拒绝服务攻击

## 功能2：上传压缩包解压循环资源占用（压缩包炸弹）

压缩包打开之后还是压缩包，无限套娃，也会造成拒绝服务攻击

在实战测试，一般套两三层测试结果就可以，不然会造成服务器宕机等影响
